# 写作或使用规范

## 分类与标签使用规范

> 写新文章前，先看这里，保持统一。

---

### 分类（categories）

只能选一个：

- 博客折腾记
- 技术笔记
- 生活随笔
- 读书笔记

---

### 标签（tags）

可选多个，尽量从下面选：

- 博客搭建
- Minima主题
- 导航模板
- Markdown
- Git
- VBA
- Python
- RPA
- 影刀
- 帆软
- BI
- kettle
- 数据分析
- 阅读书单
- AI

---

### 规则

1. 分类从表中选一个。
2. 标签从表中选 3-5 个。
3. 如果现有标签不够用，先在本文档中新增，再用到文章里。
