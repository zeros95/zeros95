# 资源导航 开发规格书

## 一、项目概述

### 1.1 项目定位
个人资源/书签分类管理工具，单文件 HTML 应用，无需后端服务。

### 1.2 核心特性
- 三级分类树结构（支持拖拽排序）
- 链接收藏功能（上限12个）
- 三种主题切换（浅色/深色/多巴胺）
- 三种展示模式（列表/卡片/随机）
- 图标自定义（180+ Font Awesome 图标）
- 数据导入/导出（JSON格式）
- 导入模式选择（增量/覆盖）
- 自动保存机制（目录选择）
- 座右铭自定义（字体大小/样式）
- 编辑授权机制
- 离线可用（本地化依赖）
- 首次使用引导弹窗

### 1.3 技术栈
| 技术 | 版本 | 用途 |
|-----|------|------|
| HTML5 | - | 单文件应用载体 |
| CSS3 | - | 样式与主题系统 |
| JavaScript | ES6+ | 业务逻辑（纯原生，无框架） |
| Tailwind CSS | Standalone | 布局框架（本地化） |
| Font Awesome | 6.5.1 | 图标库（本地化） |
| File System Access API | - | 文件操作（仅Chrome/Edge 86+） |

## 四、数据结构

### 4.1 分类树数据（treeData）
```javascript
treeData = [
  {
    id: "cat_1",           // 分类ID（自动生成）
    name: "工作相关",       // 分类名称
    icon: "fa-briefcase",   // Font Awesome 图标类名
    children: [            // 子分类数组（可选）
      {
        id: "cat_1_1",
        name: "项目管理",
        icon: "fa-folder",
        children: [],
        links: []           // 末级分类才有 links
      }
    ],
    links: []              // 非末级分类 links 为空
  }
]
```

### 4.2 链接数据结构
```javascript
link = {
  id: "link_xxx",          // 链接ID（自动生成）
  name: "GitHub",          // 链接名称
  url: "https://github.com", // URL地址
  note: "代码托管平台"      // 备注（可选）
}
```

### 4.3 收藏数据（favorites）
```javascript
favorites = [
  "cat_1_1_0",             // 格式：分类ID + "_" + 链接索引
  "cat_2_3",
  // ...最多12个
]
```

### 4.4 配置数据（config）
```javascript
config = {
  theme: "light",          // 主题：light | dark | dopamine
  openMode: "blank",       // 链接打开方式：blank | self
  greeting: "你好，小不溜", // 招呼语
  motto: "它山之石，可以攻玉。", // 座右铭内容
  mottoFontSize: 14,       // 座右铭字体大小（px）
  mottoFontFamily: ""      // 座右铭字体样式（空为默认）
}
```

### 4.5 localStorage存储
| 键名 | 内容 | 格式 |
|-----|------|------|
| `nav_tree` | 分类树数据 | JSON.stringify(treeData) |
| `nav_fav` | 收藏列表 | JSON.stringify(favorites) |
| `nav_config` | 配置数据 | JSON.stringify(config) |

## 六、功能模块

### 6.1 分类拖拽排序
- 所有级别分类支持拖拽排序
- 使用 `data-parent-id` 和 `data-depth` 属性判断同级
- 拖拽完成后自动调用 `saveData()` 和 `renderTree()`

### 6.2 导入模式选择
- **增量导入**：合并数据，同名分类合并链接（按URL去重）
- **覆盖导入**：替换数据，清空现有数据
- 导入完成后自动关闭设置面板

### 6.3 自动保存机制
- 使用 `showDirectoryPicker` 选择目录
- 数据保存到目录下的 `data.json`
- 首次选择后记住目录，直到禁用或失效
- 仅 Chrome/Edge 86+ 支持

### 6.4 座右铭功能
- 点击座右铭区域打开独立设置弹窗
- 支持字体大小调整（12-24px）
- 支持字体样式调整（宋体、黑体、微软雅黑等）
- 实时预览效果

### 6.5 编辑授权机制
- 首次编辑操作需确认授权
- 授权后后续操作静默执行
- 授权状态保存在 localStorage

### 6.6 首次使用引导
- **触发条件**：localStorage无nav_tree数据（使用DEFAULT_TREE）
- **引导内容**：建议启用自动保存，推荐选择与index.html同一目录
- **判断函数**：`isFirstUse()` 检查treeData是否与DEFAULT_TREE相同
- **记录已提醒**：localStorage存储`nav_first_use_reminded=true`
- **关闭方式**：点击"稍后设置"、点击"启用自动保存"、ESC键、点击背景
- **无痕模式**：localStorage不可用时跳过检测
- **浏览器兼容提示**：弹窗中提示仅Chrome/Edge支持自动保存

## 七、已知限制与兼容性

### 7.1 浏览器兼容性
| 浏览器 | 支持程度 | 注意事项 |
|--------|---------|---------|
| Chrome 86+ | 完全支持 | 推荐，支持所有功能 |

| Firefox | 基本支持 | 不支持 File System Access API |
| Safari | 基本支持 | 不支持 File System Access API |

### 7.2 功能限制
- 分类最多三级
- 收藏最多12个
- 删除分类需先清空链接
- 末级分类才能添加链接
- 自动保存仅 Chrome/Edge 86+ 支持

### 8.1 V2.0 详细变更
- **首次使用引导弹窗**：首次打开时弹出引导，建议设置自动保存路径
- **isFirstUse检测函数**：判断是否使用DEFAULT_TREE数据
- **loadData解析防护**：添加try-catch，防止JSON解析失败导致白屏
- **拖拽事件优化**：改用事件委托模式，避免重复绑定和内存泄漏
- **favorites索引修复**：deleteLink和批量移动后更新索引引用
- **URL特殊协议处理**：支持ftp、mailto、tel，禁止javascript协议
- **全选功能修复**：使用countLinksRecursive正确统计子分类链接
- **重复URL检测**：新增链接时检测是否已存在
- **config字段完整性**：使用Object.assign合并默认配置

### 8.2 V1.9 详细变更
- **分类拖拽排序**：所有级别分类支持拖拽排序
- **导入模式选择**：增量导入（合并数据）或覆盖导入（替换数据）
- **自动保存机制**：选择目录后自动保存 data.json
- **座右铭独立弹窗**：点击座右铭区域触发，支持字体大小和样式调整
- **设置弹窗优化**：合并招呼语和链接打开方式、移除迁移教程说明、简化自动保存说明、合并使用说明和版本信息
- **使用说明两栏布局**：内容一览无余，无需滚动
- **删除导出HTML功能**：仅保留导出JSON
- **导入JSON后关闭设置面板**
- **导入书签改名为导入HTML**
