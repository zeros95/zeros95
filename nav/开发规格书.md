# 资源导航 开发规格书

## 一、项目概述

### 1.1 项目定位
个人资源/书签分类管理工具，单文件 HTML 应用，无需后端服务。

### 1.2 核心特性
- 三级分类树结构
- 链接收藏功能（上限12个）
- 三种主题切换（浅色/深色/多巴胺）
- 三种展示模式（列表/卡片/随机）
- 图标自定义（180+ Font Awesome 图标）
- 数据导入/导出（JSON格式）
- 离线可用（本地化依赖）

### 1.3 技术栈
| 技术 | 版本 | 用途 |
|-----|------|------|
| HTML5 | - | 单文件应用载体 |
| CSS3 | - | 样式与主题系统 |
| JavaScript | ES6+ | 业务逻辑（纯原生，无框架） |
| Tailwind CSS | Standalone | 布局框架（本地化） |
| Font Awesome | 6.5.1 | 图标库（本地化） |

## 四、数据结构

### 4.1 分类树数据（treeData）
```javascript
treeData = [
  {
    id: "cat_1",           // 分类ID（自动生成）
    name: "工作相关",       // 分类名称
    icon: "fa-briefcase",   // Font Awesome 图标类名
    children: [            // 子分类数组（可选）
      {
        id: "cat_1_1",
        name: "项目管理",
        icon: "fa-folder",
        children: [],
        links: []           // 末级分类才有 links
      }
    ],
    links: []              // 非末级分类 links 为空
  }
]
```

### 4.2 链接数据结构
```javascript
link = {
  id: "link_xxx",          // 链接ID（自动生成）
  name: "GitHub",          // 链接名称
  url: "https://github.com", // URL地址
  note: "代码托管平台"      // 备注（可选，V1.6新增）
}
```

### 4.3 收藏数据（favorites）
```javascript
favorites = [
  "cat_1_1_0",             // 格式：分类ID + "_" + 链接索引
  "cat_2_3",
  // ...最多12个
]
```

### 4.4 配置数据（config）
```javascript
config = {
  theme: "light",          // 主题：light | dark | dopamine
  sidebarWidth: 240,       // 侧边栏宽度（150-500）
  viewMode: "list"         // 展示模式：list | card | random
}
```

### 4.5 localStorage存储

| `nav_tree` | 分类树数据 | JSON.stringify(treeData) |

## 五、CSS设计规范

### 5.1 CSS变量系统
所有颜色、阴影、圆角通过 CSS 变量定义，支持三种主题切换：

```css
:root {
  /* 基础色 */
  --bg-primary: #fff;
  --bg-secondary: #f5f5f5;
  --text-primary: rgba(0,0,0,0.88);
  --border-color: #d9d9d9;
  
  /* 功能色 */
  --accent: #1677ff;
  --danger: #ff4d4f;
  --success: #52c41a;
  --warning: #faad14;
  
  /* 背景渐变 */
  --bg-gradient: linear-gradient(...);
  --bg-pattern: radial-gradient(...);
  --decoration-color: rgba(...);
}
```

### 5.2 主题差异化设计
| 主题 | 背景效果 | 首页容器 | 装饰元素 |
|-----|---------|---------|---------|
| **浅色** | 晨曦渐变（蓝→白→暖） | 白色实色 + 柔和阴影 | 模糊光晕 + 线条 |
| **深色** | 夜空渐变（深蓝→紫灰） | 深灰实色 + 深阴影 | 8个星空光点 |
| **多巴胺** | 糖果渐变（粉→紫→蓝） | 白色实色 + 粉色阴影 | 彩色光晕 |

### 5.3 关键样式类
| 类名 | 用途 |
|-----|------|
| `.modal-overlay` | 弹窗遮罩层 |
| `.modal-box` | 弹窗容器 |
| `.btn` | 按钮基础样式 |
| `.btn-primary` | 主按钮 |
| `.btn-ghost` | 幽灵按钮 |
| `.fav-card` | 常用链接卡片 |
| `.fc-name-tooltip` | 常用链接名称hover提示 |
| `.tree-row` | 分类树节点行 |
| `.link-row` | 链接列表行 |
| `.link-url` | 网址（主题色，max-width: 280px） |
| `.link-note` | 备注（灰色，flex-shrink: 1，动态间距） |
| `.link-card` | 链接卡片（overflow: visible） |
| `.link-card-note-tooltip` | 卡片备注hover提示 |
| `.bg-decorations` | 背景装饰容器 |

### 5.4 链接展示模式样式
| 模式 | 复选框 | 网址样式 | 备注样式 |
|-----|--------|---------|---------|
| **列表** | 正常显示 | 主题色，max-width: 280px | 灰色文字，网址后面，动态间距 |

| 弹窗ID | 用途 | 触发方式 |
|--------|------|---------|
| `linkModal` | 新增/编辑链接 | 顶栏按钮、侧边栏按钮 |

### 6.3 弹窗关闭方式
- 点击关闭/取消按钮
- 点击弹窗外部区域

## 七、图标系统

### 7.1 图标分组定义
```javascript
const iconGroups = [
  { name: '文件夹', icons: ['fa-folder', 'fa-folder-open', ...] },
  { name: '工作商务', icons: ['fa-briefcase', 'fa-building', ...] },
  { name: '生活娱乐', icons: ['fa-shopping-cart', 'fa-plane', ...] },
  // ...共13个分组，180+图标
];
```

### 7.2 图标使用规范
- 使用 Font Awesome 6.x 类名
- 禁止使用已废弃图标：`fa-th-large`（改用 `fa-table-cells-large`）、`fa-random`（改用 `fa-shuffle`）
- 禁止使用不存在图标：`fa-target`、`fa-party-horn`
- 避免重复图标：`fa-graduation-cap`、`fa-tasks`（与 `fa-list-check` 相同）

## 八、二次开发指南

### 8.1 新增功能流程
1. **规划阶段**：确认功能需求，评估影响范围
2. **HTML层**：添加必要的 DOM 元素（弹窗、按钮等）
3. **CSS层**：添加样式，使用 CSS 变量保持主题一致性
4. **JS层**：添加函数和事件绑定
5. **测试阶段**：三种主题、两种协议（file:// 和 http://）

### 8.2 修改主题配色
修改对应主题的 CSS 变量定义：
```css
[data-theme="dark"] {
  --bg-primary: #新颜色;
  --accent: #新颜色;
  --bg-gradient: linear-gradient(...);
}
```

### 8.3 新增弹窗
1. 在 `<body>` 末尾添加弹窗 HTML
2. 添加弹窗 CSS（复用 `.modal-overlay`、`.modal-box`）
3. 添加打开/关闭函数
4. 在 ESC 快捷键处理中添加关闭逻辑
5. 添加事件绑定

### 8.4 新增图标分组
在 `iconGroups` 数组中添加新分组：
```javascript
{ name: '新分组名', icons: ['fa-icon1', 'fa-icon2', ...] }
```

### 8.5 修改收藏上限
修改以下位置：
- `toggleFav()` 函数中的 `favorites.length >= 12`
- `renderFavorites()` 中的计数显示 `(x/12)`
- 使用说明文档

## 九、已知限制与兼容性

### 9.1 浏览器兼容性
| 浏览器 | 支持程度 | 注意事项 |
|--------|---------|---------|
| Chrome | 完全支持 | 推荐 |
| Firefox | 完全支持 | - |
| Edge | 基本支持 | file:// 下 localStorage 可能受限 |
| Safari | 基本支持 | - |

### 9.2 协议兼容性
| 协议 | localStorage | 建议 |
|-----|-------------|------|
| `http://` | 正常 | 推荐 |
| `file://` | 部分浏览器受限 | 使用内存存储 fallback |

### 9.3 功能限制
- 分类最多三级
- 收藏最多12个
- 删除分类需先清空链接
- 末级分类才能添加链接
