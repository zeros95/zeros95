使用说明

分类管理
• 分类最多三级，末级分类才能添加链接
• hover 分类节点显示编辑/新增/删除按钮
• 删除分类前需先清空链接
链接管理
• 点击链接名称打开链接（新标签页）
• 点击星标按钮添加到常用（最多12个）
• 勾选链接后可批量移动或删除
展示模式
• 列表模式：紧凑列表展示
• 卡片模式：卡片式展示
• 随机模式：随机排序展示
搜索功能
• 首页搜索：快速搜索分类和链接
• 全局搜索：点击顶栏搜索按钮
数据管理
• 导出数据：保存为 JSON 文件备份
• 导入数据：从 JSON 文件恢复
• 建议定期导出备份
快捷键
• ESC：关闭所有弹窗
