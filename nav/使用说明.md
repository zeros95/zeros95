# 资源导航 使用说明

## 简介

资源导航是一个个人资源/书签分类管理工具，支持三级分类结构、多种展示模式、图标自定义等功能。

## 功能特性

### 1. 分类管理
- **三级分类结构**：大类 → 二级分类 → 三级分类（末级才能添加链接）
- **拖拽排序**：所有级别分类支持拖拽排序
- **图标自定义**：180+ Font Awesome 图标可选，按分组展示
- **新增分类**：点击侧边栏「新增」按钮或全局设置中的「新增一级分类」
- **编辑分类**：hover 分类节点显示编辑按钮，可修改名称和图标
- **删除分类**：hover 分类节点显示删除按钮（需先清空链接）

### 2. 链接管理
- **新增链接**：右上角「新增链接」按钮或侧边栏分类节点的「新增链接」
- **编辑链接**：点击链接的编辑按钮修改名称和 URL
- **删除链接**：点击链接的删除按钮
- **收藏链接**：点击星标按钮添加到常用（最多 12 个）

### 3. 批量操作
- **勾选链接**：点击链接左侧复选框
- **全选/取消全选**：点击「全选」按钮
- **批量移动**：勾选后点击「移动到」按钮，选择目标分类
- **批量删除**：勾选后点击「批量删除」按钮

### 4. 展示模式
- **列表模式**：紧凑列表展示
- **卡片模式**：卡片式展示，适合少量链接
- **随机模式**：随机排序展示，适合发现新内容

### 5. 主题切换
- **浅色主题**：默认 Ant Design 6 配色
- **深色主题**：暗色背景，适合夜间使用
- **多巴胺主题**：活力配色

### 6. 座右铭功能
- **自定义座右铭**：点击首页座右铭区域，打开设置弹窗
- **字体大小**：可选 12px-24px
- **字体样式**：可选宋体、黑体、微软雅黑、楷体、仿宋、Arial、Georgia
- **实时预览**：修改后立即生效
- **默认座右铭**：它山之石，可以攻玉。

### 7. 搜索功能
- **首页搜索**：快速搜索链接
- **全局搜索**：点击顶栏搜索按钮，搜索所有链接

### 8. 数据管理（详见使用说明）
- **导出JSON**：设置中点击「导出JSON」按钮，保存当前数据为JSON文件
- **导入JSON**：设置中点击「导入JSON」按钮，从JSON文件恢复数据
  - **增量导入**：合并数据，同名分类合并链接（按URL去重）
  - **覆盖导入**：替换数据，清空现有数据
- **导入HTML**：设置中点击「导入HTML」按钮，支持Chrome/Edge/Firefox书签HTML文件（详见使用说明）
- **自动保存**：启用后选择目录，数据自动保存到目录下的 data.json
- **首次使用引导**：首次打开时弹出引导弹窗，建议设置自动保存路径

### 9. 编辑授权
- **授权机制**：首次编辑操作（新增、修改、删除）需确认授权
- **授权后静默**：授权确认后，后续编辑操作全程静默，不再重复提示
- **授权状态持久**：授权状态保存在浏览器，刷新页面后保持

### 10. 侧边栏
- **折叠/展开**：点击分类节点左侧的箭头图标

## 使用方式

### 方式一：直接打开文件
- 双击 `index.html` 打开
- 需要网络加载 Tailwind CSS 和 Font Awesome（已本地化）
- Edge 浏览器可能限制 localStorage，数据不会持久保存

### 方式二：HTTP 服务器访问（推荐）
```bash
# Python
python -m http.server 8080

# 访问 http://localhost:8080/index.html
```

### 方式三：分享给他人
- 打包整个文件夹（包含 `lib/` 目录）
- 对方直接打开 `index.html` 即可使用

## 数据存储

- 使用 localStorage 存储（键名：`nav_tree`, `nav_fav`, `nav_config`）
- `file://` 协议下部分浏览器限制 localStorage
- 建议定期导出数据备份

## 数据迁移教程

### 首次使用引导
- **触发条件**：首次打开工具（localStorage无用户数据）时自动弹出引导弹窗
- **引导内容**：建议启用自动保存，推荐选择与index.html同一目录
- **操作选项**：
  - **启用自动保存**：选择目录后启用自动保存功能
  - **稍后设置**：关闭弹窗，可在设置中手动启用
- **不再提醒**：关闭弹窗后记录已提醒状态，后续不再弹出
- **注意事项**：
  - 仅Chrome/Edge浏览器支持自动保存功能
  - 无痕模式下localStorage不可用，不显示引导弹窗

### 方式一：从本工具迁移数据
1. **导出数据**：设置 → 导出JSON → 保存 `data.json` 文件
2. **导入数据**：新设备打开工具 → 设置 → 导入JSON → 选择 `data.json` 文件
3. **导入模式**：
   - 增量导入：合并数据，同名分类合并链接
   - 覆盖导入：替换数据，清空现有数据

### 方式二：从浏览器书签导入
1. **导出浏览器书签**：
   - Chrome：书签管理器 → 右上角三点 → 导出书签
   - Edge：收藏夹 → 右上角三点 → 导出收藏夹
   - Firefox：书签 → 管理书签 → 导入和备份 → 导出书签为HTML
2. **导入书签**：设置 → 导入HTML → 选择导出的HTML文件 → 选择导入模式
3. **导入模式**：
   - 增量导入：合并数据，同名分类合并链接
   - 覆盖导入：替换数据，清空现有数据
4. **注意事项**：
   - 书签层级最多保留三级
   - 超过三级的链接合并到末级分类

### 方式三：迁移到新设备
1. 打包整个文件夹（包含 `index.html` 和 `lib/` 目录）
2. 复制到新设备
3. 打开 `index.html` 即可使用
4. 如需恢复数据：导入之前导出的 `data.json`

## 快捷键

- `ESC`：关闭所有弹窗
- `Enter`：在输入框中按回车保存

## 注意事项

1. **末级分类才能添加链接**：有子分类的分类节点不能直接添加链接
2. **删除分类前需清空链接**：分类下有链接时无法删除
3. **收藏上限 12 个**：常用链接最多 12 个，首页显示更美观
4. **URL 格式**：输入 URL 时无需添加 `http://`，系统自动处理
5. **自动保存仅 Chrome/Edge 支持**：Firefox/Safari 不支持 File System Access API
